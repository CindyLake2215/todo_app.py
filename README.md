This application is a web-based interface built with Streamlit. It shows a list of popular movies, each with its own rating and release date. Users can interact with it by sliding the bar in either direction to see the lowest rating and highest rating, 10 showing none as no movie within the rating system I am using has a perfect 10.  

The use cases I implemented are display of movie data, users can see a list of movies with details about their ratings and release years, a filter by ratings option so that they can decide which movies to watch, and it can help movie critics.
